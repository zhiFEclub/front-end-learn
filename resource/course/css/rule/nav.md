# CSS 常用属性概览
建议学习时长： 10分钟  
学习方式：了解  

## 学习目标
* 大概了解下，CSS 有哪些常用属性。

CSS 的属性非常之多，我们只需了解有哪些即可。很多属性，不知怎么用，查文档好了。
