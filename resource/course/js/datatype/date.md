# JavaScript内置对象之日期(Date)
建议学习时长： 30分钟  
学习方式：了解  

## 学习目标
* 了解日期的常用方法

## 详细介绍
`Date` 对象是 `JavaScript` 提供的日期和时间的操作接口。

日期这块只需了解。用的时候来查阅即可。对于复杂的日期操作，可以用第三方库[Moment.js](http://momentjs.com/)。
