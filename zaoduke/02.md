# 知行社的前端早读课 第2期
## wtfjs
### 背景
众所周之，JavaScript 有很多坑，或者说是奇怪的特性。从坑里爬出来后，可能就想 WTF（Want to f*ck） XD。

### 概要
这是一系列的 JavaScript 语言的有(hei)趣(keji)技巧的合集，如
```
[] == ![] // -> true
```

又如
```
!![]       // -> true
[] == true // -> false
```

项目地址：https://github.com/denysdovhan/wtfjs
