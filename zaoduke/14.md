## 《咦？浏览器都能做人脸检测了？》
### 概要
介绍了做人脸检测的几种方式：
* Chrome 56+ 可以直接用浏览器内置 API
* 低版本浏览器可以用第三方库 [tracking.js](https://trackingjs.com/) 和 jquery.facedetection。
* Node.js 可用 node-opencv 模块。

当然目前用浏览器识别的情况，从准确率，处理速度，功能性上来说并不乐观。但随着基础设施(硬件、浏览器、工具、库等)的逐渐增强和完善，情况会越来越好的，
